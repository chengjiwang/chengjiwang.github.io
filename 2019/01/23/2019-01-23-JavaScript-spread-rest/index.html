ES6 - spread operator 與 rest operator

展開運算子(spread operator)

  • 展開運算子會將陣列裡面的值一個一個取出來，並 return 回去

  • 展開可以將類陣列轉成陣列 :
    類陣列的原型(__proto)和陣列的原型不同，只有部分的方法。

  • 使用展開可以避免陣列因為傳參考而影響到原本的陣列

其餘運算子 (rest operator，也可稱為其餘參數)

  • 其餘參數會將其餘的值轉為陣列，用在函式定義中的傳入參數或解構賦值時。
