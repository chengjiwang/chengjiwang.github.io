箭頭函式的寫法

箭頭函式的 this

箭頭函式沒有自己的情境，箭頭函式的 this 依宣告時候的環境而決定。
如果在全域狀態下宣告箭頭函式，內部的 this 就是 window 。

要使 setTimeout 函式的 this 為 car 物件的話，在以前我們可以使用這個方法，
將 this 賦值給新的變數，然後在 setTimeout 函式 內使用新的變數。

更簡便的方法是，使用 arrow function

如果直接將物件的函式使用箭頭函式，則會變成 window 物件
