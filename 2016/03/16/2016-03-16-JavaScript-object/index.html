JavaScript - 物件

物件是屬性(property)所構成的一個集合。每個屬性皆具有一個名稱(name)與一個值(value)。

一. 物件字面(object literal)

建立物件

為物件加上一個變數宣告，然後在 { } 裡加入屬性，屬性間需以逗號分開。

如何操作屬性:

  1. 存取(access)物件的屬性
  1. 如何改變一個屬性
    將一個新值賦值給該屬性。
  1. 如何添加新的屬性
    指定新的屬性並對它賦值
  1. 刪除屬性
    使用 delete。 屬性刪除成功，delete 運算式會傳回 true。

傳參考(pass-by-reference)

物件是以 reference 傳遞，所以當你把一個物件被賦值給一個新的變數時，你所傳遞的是物件 reference，這代表著，如果你改變新物件的屬性，會因而改變原本物件的屬性。

物件的方法(method)

物件裡的函式稱為方法(method)，如果要呼叫物件的方法，可使用 dog.bark();

二. 物件建構程序(object constructor)

當有大量的物件須建立時，我們可以使用另一種方法建立物件，就是物件建構程序(object constructor)。

建構程序的過程分為兩個步驟:

  1. 首先定義建構程序函式(名稱通常以大寫開頭)。
  2. 接著用它來建立物件。

建構程序函式引數改成物件字面

instanceof

instanceof 可用於確定某個物件是否為特定建構程序的一個實例。
